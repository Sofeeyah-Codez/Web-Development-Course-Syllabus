Course Duration: 12 Weeks

Modules

HTML Fundamentals
  • Introduction to HTML
  • HTML Elements & Tags
  • Forms and Inputs
  • Tables and Lists
Objective:
Learn the structure of web pages and create basic HTML documents.
Assignment:
Build a personal profile page using headings, paragraphs, images, and lists.
CSS Styling
  • CSS Syntax and Selectors
  • Box Model and Positioning
  • Flexbox and Grid Layouts
  • Colors, Fonts, and Backgrounds
Objective:
Style web pages with modern CSS techniques for layout and design.
Assignment:
Create a landing page with responsive design using Flexbox or Grid.
JavaScript Basics
  • Variables and Data Types
  • Functions and Loops
  • DOM Manipulation
  • Events and User Interaction
Objective:
Add interactivity to web pages using JavaScript.
Assignment:
Create a to-do list app that allows adding, editing, and removing tasks.
Advanced JavaScript & Projects
  1. ES6 Features (let, const, arrow functions)
  2. Fetch API and JSON
  3. Form Validation
  4. Small Projects: Quiz App, Weather App
Objective:
Develop dynamic web applications and practice advanced JavaScript features.
Assignment:
Build a small interactive web project using HTML, CSS, and JavaScript.

Weekly Plan

Week 1-2: HTML Basics

  1. Introduction to HTML
  2. Creating Your First Page

Week 3-4: CSS Fundamentals

  1. Selectors and Properties
  2. Layouts with Flexbox & Grid

Week 5-7: JavaScript Basics

  1. Variables and Functions
  2. Events and DOM Manipulation

Week 8-12: Projects & Advanced Topics

  1. Mini Projects: Quiz App, Calculator
  2. Introduction to APIs and Fetch
